Educreations Embed Gadget
=========================

### Basic use case:

-   Authors can embed Educreations videos inside the course

    -   Authors paste the iframe embed code

    -   Authors can also adjust the embed code if necessary

### To compile stylus to css

    npm install
    grunt

### To test

    npm install
    grunt
